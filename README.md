# SDAICore

	## [1.0.0.0-SNAPSHOT] 
	
		### FEATURE
			- Creación de la clase BeanIndex
				- Esta clase se encargara de llevar a cabo todas las acciones necesarias para cargar la pagina inicial de la aplicación 
				que se este ejecutando.
			- Creación del fichero faces-config.txt
				- Este fichero guardara toda la información relativa a las definiciones de los beans de core, para cuando se desarrolle 
				una nueva aplicación poder copiarla en el fichero faces-config.xml de la nueva aplicación.
